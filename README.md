## Iot-Fire-Early-Warning-System
an IoT-based fire early warning system using the ESP32 DevKit V1 to detect potential fire hazards through real-time environmental monitoring. The system combines smoke, flame, and temperature sensing with instant Telegram notifications to enable rapid response, even when users are away from the monitored location.
## Features
- Smoke Detection
- Flame Detection
- Temperature Monitoring
- Telegram Notification
- LCD Monitoring
- WiFi Communication
## Hardware
- ESP32 DevKit V1
- MQ-2
- KY-026
- DS18B20
- LCD I2C
- Buzzer
## Software
- Arduino IDE
- Telegram Bot
- WiFi
- OneWire
-DallasTemperature
